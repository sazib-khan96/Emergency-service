# Emergency Service App 🚨

A simple web app for calling emergency services with coin system, copy feature, and call history.

---

## 🚀 Features

- 📞 Call emergency numbers
- 💰 Coin deduction system (10 coins per call)
- 📋 Copy number to clipboard
- ❤️ Heart counter feature
- 🕒 Call history tracking
- 🗑️ Clear history button

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (Vanilla JS)
